# Trash2Cash Assets

## Logo Usage

### Main Logo
- **File**: `logos/trash2cash_logo.png`
- **Format**: PNG (Portable Network Graphics)
- **Colors**: 
  - Primary Green: `#72BF44`
  - White background for center circle
- **Usage**: Main navigation, headers, official documents

### Logo Description
The Trash2Cash logo features a recycling symbol with three curved arrows forming a circle, representing the circular economy and waste-to-cash concept. The center contains a dollar sign ($) symbolizing the monetary value derived from recycling activities.

### Implementation
```jsx
import logoPng from '../assets/images/logos/trash2cash_logo.png';

<img 
  src={logoPng}
  alt="Trash2Cash Recycling Logo"
  className="h-8 w-8"
/>
```

### Brand Guidelines
- Maintain aspect ratio when resizing
- Ensure sufficient contrast against background
- Use white circular background for optimal visibility on colored backgrounds
- Minimum size: 24x24px for web usage
