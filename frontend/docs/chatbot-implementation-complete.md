## 🧠 Knowledge Base Content

### Platform Information
- **Name**: Trash2Cash.lk
- **Mission**: Smart Waste Collection Network for Sri Lanka
- **Location**: Colombo, Sri Lanka
- **Architecture**: MERN Stack (MongoDB, Express.js, React.js, Node.js)

### User Roles Coverage (5 Stakeholders)
1. **Customers** (Households & Businesses) - Waste generators
2. **Pickup Agents** (Drivers/Collectors) - Physical collection
3. **Pickup Partners** (Licensed Collectors) - Fleet management
4. **Recyclers** (End-buyers) - Material processing
5. **Administrators** (Platform operators) - System management

### System Modules (5 Core Areas)
1. **Bin Management** - IoT smart bins with ultrasonic sensors
2. **Finance Management** - Stripe payments, automated commissions
3. **Agent Management** - Performance tracking, route optimization
4. **Waste Management** - Inventory dashboard, bulk ordering
5. **System Management** - User roles, pricing control, configuration

### Technical Specifications
- **IoT**: ESP32 microcontrollers with ultrasonic sensors
- **Authentication**: JWT tokens for security
- **Payments**: Stripe API integration
- **Mapping**: Google Maps API for routing
- **Development**: Agile methodology
